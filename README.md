# Inventory Daily Registration System

A comprehensive web application for tracking and managing daily inventory registrations for chat exporter companies. Built with Node.js, React, and PostgreSQL.

## Features

- **Daily Inventory Registration**: Log daily inventory activities with timestamps
- **Chat Export Tracking**: Monitor quantities of chat export files/records
- **Storage Management**: Track and visualize storage consumption
- **User Licensing**: Manage user accounts and licenses
- **API Quota Monitoring**: Track API usage and quotas
- **Category Management**: Organize exports by different categories
- **Analytics & Reporting**: View detailed reports and analytics
- **User Management**: Role-based access control (Admin, Manager, User)
- **Real-time Dashboard**: Live inventory metrics and statistics

## Tech Stack

### Backend
- **Runtime**: Node.js (v16+)
- **Framework**: Express.js
- **Database**: PostgreSQL
- **Authentication**: JWT
- **Validation**: Joi
- **Logging**: Winston

### Frontend
- **Framework**: React (v18+)
- **State Management**: Redux
- **Styling**: Tailwind CSS
- **Charts**: Chart.js / Recharts
- **HTTP Client**: Axios
- **Form Handling**: React Hook Form
